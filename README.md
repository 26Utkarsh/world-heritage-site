# AI-Powered Site Explorer

A modern, responsive web application for exploring sites with integrated map views, image galleries, and an AI-powered conversational guide.

## Features

- **Interactive Map View**: Explore physical site locations visually using Leaflet maps.
- **AI Guide**: Ask questions about the sites and get AI-generated insights (powered by Gemini).
- **Image Gallery**: Browse rich visual galleries of the locations.
- **Search & Pagination**: Easily filter and navigate through numerous sites.
- **Mobile-Responsive**: Fully optimized for both Desktop and Smartphone viewports.
- **Modern UI**: Styled with Tailwind CSS, bringing a clean, accessible layout with dark mode support.

## Tech Stack

- **Frontend Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **Maps**: Leaflet (`react-leaflet`)
- **Animation**: Motion (`motion/react`)
- **AI/LLM**: Google GenAI SDK (`@google/genai`)
- **Build Tool**: Vite
